# یک برنامه ساده یوز برای اندروید

این فقط یک نمونه برنامه ساده اندرویدی برای ساخت چت بات با کتابخونه قدرتمند یوز میباشد


**اجرا**

```bash
git clone https://github.com/yooz-lang/simple-yooz-android
cd simple-yooz-android
flutter pub get
flutter run
```
